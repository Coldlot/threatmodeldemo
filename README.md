# ThreatModelDemo

Демонстрационное iOS-приложение (SwiftUI), созданное для отработки навыков **Application Security (AppSec)**.

Проект является частью учебного плана по переходу из iOS-разработки в AppSec и используется как полигон для:
- Построения модели угроз (Threat Modeling)
- Статического анализа кода (SAST)
- Внедрения безопасности в CI/CD (GitHub Actions, Semgrep, CodeQL)
- Настройки правил блокировки слияния (Branch Protection Rules)

## Ключевые компоненты

| Компонент | Назначение | Статус |
|-----------|------------|--------|
| **Jailbreak Detection** | Симуляция проверки, запущено ли приложение на взломанном устройстве | Обходится через Frida (на симуляторе) |
| **SSL Pinning** | Упрощённая проверка сертификата для защиты от MITM-атак | Требует реального устройства для тестирования |
| **Логин-форма** | Сохранение токена в Keychain для демонстрации безопасного хранения | Работает |
| **Hardcoded Certificate** | Сертификат, найденный Semgrep и удалённый из кода | Исправлено |

## CI/CD и безопасность

В проекте настроен пайплайн безопасности (`.github/workflows/security.yml`), который автоматически запускается на каждый Pull Request в `main` или `develop`.

### Инструменты проверки

| Инструмент | Тип анализа | Описание |
|------------|-------------|----------|
| **Semgrep** | SAST (кастомное правило) | Поиск хардкод-секретов: паролей, API-ключей, токенов, сертификатов |
| **CodeQL** | SAST (поток данных) | Поиск логических уязвимостей в Swift-коде |
| **GitGuardian** | Secret Scanning | Обнаружение секретов в коммитах (Generic High Entropy) |

### Находки Semgrep

Было написано кастомное правило `.semgrep/custom-rules.yaml` для поиска строк, присвоенных переменным с подозрительными именами:

- `password`, `passwd`, `pwd`, `secret`
- `token`, `api_key`, `apiKey`
- `db_password`
- `cert`, `certificate`

**Пример находки:** зашитый SSL-сертификат (`let pinnedCertString = "-----BEGIN CERTIFICATE-----..."`) был обнаружен и удалён из кода.

## Branch Protection

Настроены правила защиты ветки `main`: слияние PR заблокировано, если проверки безопасности (Semgrep SAST, CodeQL Analysis) не пройдены.

## Структура проекта
```
ThreatModelDemo/
├── .github/
│ └── workflows/
│ └── security.yml # Пайплайн безопасности (Semgrep + CodeQL)
├── .semgrep/
│ └── custom-rules.yaml # Кастомное правило поиска секретов
├── ThreatModelDemo/
│ ├── ContentView.swift # Основной экран + Jailbreak Detection
│ ├── LoginView.swift
│ └── KeychainManager.swift
└── ThreatModelDemo.xcodeproj/
```


## Как запустить

1. Клонировать репозиторий.
2. Открыть `ThreatModelDemo.xcodeproj` в Xcode.
3. Выбрать симулятор и нажать Run (⌘R).

## Требования

- Xcode 15+
- iOS 17+ Simulator
- Swift 5.9+
